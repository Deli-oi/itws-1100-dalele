Flower Facts Quiz is a full-stack web app where users enter their name, answer 10 randomly served flower trivia questions, and have their score saved to a MySQL leaderboard — built on a PHP backend that reads questions from and writes scores to a database, with all user input handled through prepared statements. The frontend is a clean, single-page jQuery experience that shuffles answer options, gives instant right/wrong feedback, and pulls the live leaderboard after each run, all deployed to an Apache server on Azure.
